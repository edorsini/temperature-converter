# temperature-converter

This converts temperatures as follows:
  * Celcius => Fahrenheit
  * Fahrenheit => Kelvin
  * Fahrenheit => Celcius
  * Celcius => Kelvin

This includes the use of Mocha and Chai in order to test the conversions:
